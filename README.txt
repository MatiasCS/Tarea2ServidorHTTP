Consideraciones:

-Para ejecutar la tarea se necesita netbeans, con JDK actualizado a la última version.

-El cliente que no este ejecutando el servidor TCP debe modificar la IP a la que el cliente
TCP se conecta, reemplazandola por la direccion IP del host que corre dicho servidor.

-Para enviar un archivo se necesita que se ubique en la carpeta 
Tarea2ServidorHTTP\ServidorHttp\Enviar correspondiente al cliente que enviará ese archivo.

-Al momento de querer recibir un archivo se requiere que exista un archivo dirigido al solicitante.
